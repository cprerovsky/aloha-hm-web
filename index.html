Edit Better.

Aloha Editor provides an intuitive, MS Word-like editing experience, embedded seamlessly in your web application.

Try Aloha Editor

Give it a quick try - you'll like it. Try shortcuts like

  • ctrl+b to make something bold
  • ctrl+i to make text italic
  • and see how the caret reflects the current styling

which also works for colored text.

MS-Word friendly. As you expect it to be.

Most of the text on the web is still created using MS Word. We think, it should be written with Aloha Editor. Still, if you want to (or have to) use it, we made sure you can: just paste. We'll take care of the rest.

Open Source. Take a good, hard look.

What's mine is yours. Want to know how we created the most advanced editor? Take a look - it's open source! Want to use it on an open source project of your's? No problemo.

Functional WYSIWYG. For the hackers.

Aloha Editor follows the functional programming paradigm. Not keeping state means you simply hook into the editor stack and use our extensive API to create a unique editing experience for your customers.
