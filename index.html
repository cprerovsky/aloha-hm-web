Edit Better.

Aloha Editor provides an intuitive, MS Word-like editing experience, embedded seamlessly in your web application.

More

Functional WYSIWYG.
For the hackers.

Aloha Editor follows the functional programming paradigm. Keeping state out of the API enables you to simply hook into the editor stack and create a unique editing experience for your customers.

MS-Word friendly.
As you expect it to be.

Most of the text on the web is still created using MS Word. We think, it should be written with Aloha Editor. Still, if you want to (or have to) use it, we made sure you can: just paste. We'll take care of the rest.

60fps.
Ridiculously smooth.

Our stack executes in 5-6ms* for most editing operations, with more expensive operations touching the 17ms mark.

* Yep, that's a whopping 200fps. Tested on a MacBook Pro, Core i5 with Google Chrome.
